# ShakespeareGPT

This project is an Ollama-powered chatbot that takes questions about Shakespeare and his plays and sonnets. It uses embeddings created from public domain documents of his works to answer questions, and can also lookup general facts about Shakespeare and his life.

`shakespeare_docs` holds the embeddings, which were created by `store_docs.py`.

## How to Use (WIP)

You will need to download Ollama before using this project. 

To use this project, download the Github repo to your local machine, then install all the imports in the various .py files. You will then be able to run the project from main.py. The chatbot interface will let you ask questions related to Shakespeare. Please note that the model takes time to respond; you may need to wait up for a minute or two to get a response.
